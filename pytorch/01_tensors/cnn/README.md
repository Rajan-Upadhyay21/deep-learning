# CNN in PyTorch

This folder contains Python programs focused on **Convolutional Neural Networks (CNNs)** using **PyTorch**.

CNNs are one of the most important deep learning architectures for working with image data. They are widely used in image classification, object detection, medical imaging, face recognition, self-driving systems, and many other computer vision applications.

The purpose of this folder is to provide practical CNN examples in a simple and structured way. These files help build understanding of convolution layers, pooling, dropout, batch normalization, augmentation, training loops, and evaluation.

## Topics Covered

- CNN for MNIST
- CNN for CIFAR-10
- binary image classification
- dropout in CNN
- batch normalization in CNN
- image augmentation
- custom CNN architecture
- CNN training loop
- CNN evaluation
- feature map visualization
