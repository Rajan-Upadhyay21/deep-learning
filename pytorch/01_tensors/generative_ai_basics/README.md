# Generative AI Basics in Python

This folder contains Python programs focused on **generative AI basics**, which are important for understanding how modern AI systems create text, images, and other forms of content.

Generative AI refers to AI systems that can generate new outputs such as text, code, summaries, answers, images, embeddings, and structured responses based on patterns learned from data. It is one of the most important areas in modern artificial intelligence and powers many practical applications such as chatbots, writing assistants, image generators, code assistants, search copilots, and enterprise AI tools.

The main purpose of this folder is to provide practical Python examples that introduce the basic building blocks of generative AI workflows in a simple and structured way. These examples help build understanding of prompt engineering, sampling strategies, retrieval-augmented generation, embeddings, simple agent workflows, postprocessing, and end-to-end pipeline thinking.

This folder is designed to help learners understand how generative AI systems are structured before moving into more advanced projects such as LLM applications, multimodal AI systems, AI agents, and production deployment.
