# Deep Learning Basics in Python

This folder contains Python programs focused on **deep learning basics**, which are important for understanding how modern neural networks learn from data.

Deep learning is a branch of machine learning based on neural networks with multiple layers. These networks can automatically learn complex patterns from data and are widely used in computer vision, natural language processing, recommendation systems, speech recognition, generative AI, and many other modern AI applications.

The main purpose of this folder is to provide practical Python examples that introduce the foundational ideas behind deep learning in a simple and structured way. These examples help build understanding of perceptrons, multilayer neural networks, forward propagation, backpropagation, activation functions, loss functions, optimization, dropout, batch normalization, and basic model training.

This folder is designed to help learners understand the core concepts behind deep learning before moving into more advanced topics such as CNNs, RNNs, transformers, and large language models.
